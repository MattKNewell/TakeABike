

## WiCS Spring 2019 - Take A Bike!
---
**Description**
A web app to help car communtors decide to take a bike instead!

---

To run this locally:
#### Set up your loval environment
1. `virtualenv env; source env/bin/activate; pip install -r requirements.txt`

#### Run the app!
2. `python main.py`
